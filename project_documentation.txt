## 2. SEO Implementation Details

### 2.1 Meta Tags & Head Elements
- **Title Tags**: Unique, keyword-optimized `<title>` tags for every page (e.g., "About Us | Huntington Park Spider Control | (555) 555-5555").
- **Meta Descriptions**: Compelling `<meta name="description">` summaries to improve click-through rates on search engines.
- **Robots Meta**: `<meta name="robots" content="index, follow">` ensuring search engines crawl and index the pages.
- **Canonical URLs**: `<link rel="canonical" href="...">` pointing to the definitive URL for each page to prevent duplicate content issues.
- **Google Site Verification**: Tag included for Google Search Console ownership.

### 2.2 Open Graph (Social SEO)
Used to control how URLs look when shared on social media and messaging apps:
- `og:title`, `og:description`, `og:type` (website), `og:url`, `og:image` (optimized WebP image), and `og:site_name`.

### 2.3 Schema.org Structured Data (JSON-LD)
Rich snippets added directly in the `<head>` of HTML files to help Google understand the business entity:
- **LocalBusiness Schema**: Defines the business name, address, geo-coordinates, price range, and opening hours.
- **BreadcrumbList Schema**: Defines the site hierarchy (Home > About Us, etc.) so Google can display breadcrumb navigation in search results.
- **FAQPage Schema** (where applicable): Highlights frequently asked questions directly in search results.

### 2.4 Technical SEO & Performance
- **Semantic HTML**: Proper use of `<header>`, `<main>`, `<section>`, `<article>`, and `<footer>` tags.
- **Heading Hierarchy**: Strict use of a single `<h1>` per page, followed by logical `<h2>` and `<h3>` tags for content structure.
- **Image Optimization**: Use of modern WebP formats, compressed sizes, and `alt` tags for accessibility and image search ranking.
- **Performance**: Preconnecting to Google Fonts (`rel="preconnect"`) and loading Tailwind CSS efficiently. External scripts (like `main.js`) are loaded with the `defer` attribute so they don't block HTML parsing.

### 2.5 Design & User Experience (UX) SEO Factors
- **Mobile Responsiveness**: Built completely mobile-first using Tailwind CSS. Elements resize, wrap, or hide appropriately based on screen size (e.g., the centered 5-card grid on the About page).
- **Legibility**: Use of sans-serif fonts (Inter) for body text and serif fonts (Roboto Slab) for headings. Adjustments made to letter-spacing and line-height for optimal reading experience.
